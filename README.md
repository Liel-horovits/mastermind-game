# Mastermind Game 
**A logic-based code-breaking web game built with Vanilla JavaScript.**

## Overview
An interactive version of the classic "Bulls and Cows" game. The player must guess a hidden 4-color code within a limited number of attempts based on the chosen difficulty.

## Key Features
- **Drag & Drop:** Fully interactive UI using HTML5 Drag and Drop API.
- **Game Logic:** Accurate scoring algorithm (Bulls vs. Hits) with duplicate prevention.
- **Difficulty Levels:** Supports Easy, Medium, and Hard modes via URL parameters.
- **Multimedia:** Integrated background music, sound effects, and confetti animations.

## Technical Skills Demonstrated
- **State Management:** Tracking attempts, guesses, and game status.
- **DOM Manipulation:** Dynamically creating game boards and feedback elements.
- **ES6+ JavaScript:** Using modern syntax for clean, maintainable code.

## How to Run
Simply open `homepage.html` in your browser.
